## 📊 Telemetry & Governance

### The Frustration Index (FI)
To prevent "Logic Thrashing" (infinite loops), HHaiS implements the **Frustration Index**. If the dialectic tension between the Greek and Japanese agents fails to produce a semantic delta over 3 iterations, the FI triggers a "Circuit Breaker" to force a conclusion.
